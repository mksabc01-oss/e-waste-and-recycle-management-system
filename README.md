# E-Waste and Recycle Management System

An e-commerce and management web application built for handling e-waste collection, buying, and recycling process.

##  Features
* **User Portal:** Browse recycled products, add to cart, and checkout.
* **Stock & Inventory:** Manage e-waste categories and stock levels.
* **Payment Integration:** Simple payment interface for users.
* **Authentication:** User login and signup functionality.

##  Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript
* **Backend:** PHP
* **Database:** MySQL
* **Server:** XAMPP / Apache
